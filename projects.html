Boulder Blast

Boulder Blast is a 2D arcade game developed as a part of CS32 (Introduction to Data Structures & Algorithms) taken at UCLA. This project focused on object oriented design and object management. The specification provided by the instructor for the project can be seen here. What I specifically implemented was the following files (repository):

  • Actor.h
  • Actor.cpp
  • Studentworld.h
  • Studentworld.cpp
Language Used:
  • C++
Frameworks/Tools Used:

  • OpenGL
Concepts:
  • Object Oriented Design
  • Dynamic Memory/Object Management
  • Basic AI
